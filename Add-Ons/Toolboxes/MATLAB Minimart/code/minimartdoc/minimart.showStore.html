minimart.showStore

Show store

collapse all in page

Syntax

  • minimart.showStore(store)
    example
  • minimart.showStore(store,options)
    example

Description

example

minimart.showStore(store) shows the store at location store.

example

minimart.showStore(store,options) passes the extra inputs options to web.

Input Arguments

store — Store folder
char array

The path to the store folder.

options — Web options
varargin

Additional inputs to be passed on to web.
