Creating a store

To create a store:

  1. Create a shared file system location accessible to users in your MATLAB community
  2. Deposit toolbox files (.mltbx files) in the shared location
  3. Build the store using minimart.buildStore
  4. Check the store using minimart.showStore
  5. Publicize the location of the store to your MATLAB community
  6. When toolboxes are added, rebuild the store

MATLAB recognizes that .mltbx files generated from the same .prj file are different versions of the same toolbox, rather than different toolboxes..

Browsing and installing toolboxes from a store

To view a store, use minimart.showStore. The store home page lists all the toolboxes in the store.

Each toolbox entry includes a "more info" button to browse details for each version.

Each toolbox page contains a version dropdown and a "download" button. In the MATLAB help browser, the button initiates the installation of the toolbox. In other browsers, the button initiates the download of the toolbox.

Updating toolboxes from the store

You can use minimart.updateToolboxes to update your installed toolboxes to the latest versions from a store. There are three update modes available:

  • Automatic, where toolboxes are automatically upgraded to the latest available versions
  • Manual, where you are prompted to upgrade or downgrade toolboxes to the latest available versions
  • Query, where a status report is generated

You may wish to create a shortcut on the MATLAB desktop to provide toolbox updates at the push of a button using minimart.updateToolboxes(store,'auto').
