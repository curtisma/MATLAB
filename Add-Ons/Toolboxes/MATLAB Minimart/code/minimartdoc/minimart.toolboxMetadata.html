minimart.toolboxMetadata

Query metadata of toolbox

collapse all in page

Syntax

  • s = minimart.toolboxMetadata(toolbox)
    example

Description

example

s = minimart.toolboxMetadata(toolbox) returns a struct s containing metadata for the toolbox with filename toolbox.

Input Arguments

toolbox — Name of toolbox file
char array

Name of the toolbox file (.mltbx file), specified as a character vector. The name includes the relative or absolute path to or the URL of the file.

Output Arguments

s — Toolbox metadata
struct

Full metadata for the toolbox. The struct contains the fields Filename, Name, Version, Guid, Author, Contact, Organization, Summary, Description and Screenshot.
