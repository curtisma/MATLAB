Distribution

MATLAB Minimart is distributed as a MATLAB toolbox file (.mltbx file) via the MATLAB File Exchange.

System requirements

MATLAB Minimart requires MATLAB version R2016a or later.

Sites generated by MATLAB Minimart have been tested with the MATLAB help browser and with Google Chrome.
