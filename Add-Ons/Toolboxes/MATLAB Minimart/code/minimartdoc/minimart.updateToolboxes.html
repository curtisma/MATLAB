minimart.updateToolboxes

Update toolboxes from store

collapse all in page

Syntax

  • minimart.updateToolboxes(store)
    example
  • minimart.updateToolboxes(store,mode)
    example

Description

example

minimart.updateToolboxes(store) updates toolboxes from the folder store in manual mode.

example

minimart.updateToolboxes(store,mode) updates toolboxes from the folder store in the mode mode.

Input Arguments

store — Store folder
char array

The path to the store folder.

mode — Update mode
manual (default) | auto | query

Update mode, specified as manual, auto or query. In manual mode, the user is prompted to confirm upgrades and downgrades. In auto mode, only upgrades are applied. In query mode, the user is informed of available upgrades and downgrades and also of current and missing items.
