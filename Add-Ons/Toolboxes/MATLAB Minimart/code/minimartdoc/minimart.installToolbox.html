minimart.installToolbox

Install toolbox file

collapse all in page

Syntax

  • minimart.installToolbox(toolbox)
    example
  • minimart.installToolbox(toolbox,license)
    example
  • minimart.installToolbox(toolbox,license,silent)
    example

Description

example

minimart.installToolbox(toolbox) installs the toolbox file (.mltbx file) specified by toolbox.

example

minimart.installToolbox(toolbox,license) accepts the license agreement before installing the toolbox if license is true.

example

minimart.installToolbox(toolbox,license,silent) installs silently if silent is true, and presents a confirmation dialog if silent is false.

Input Arguments

toolbox — Name of toolbox file
char array

Name of the toolbox file (.mltbx file), specified as a character vector. The name includes the relative or absolute path to or the URL of the file.

license — Indicator to accept license agreement
false (default) | true

Indicator to accept the license agreement, specified as true or false. By default, license is false. If the toolbox does not have a license agreement, the value of license has no effect on installation.

silent — Indicator to install silently
true (default) | false

Indicator to install silently, specified as true or false. If silent is false, the user must click through a confirmation dialog to install the toolbox. By default, silent is false.
